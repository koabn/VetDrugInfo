Как пользоваться приложением

  • Введите полное название препарата (например, амоксициллин) для наиболее точного поиска.
  • Поиск ведётся по двум базам: с дозировками, оформление может быть временно неидеальным и без таблиц, но всегда читаемо.
  • В результатах поиска напротив каждого препарата отображается цветной кружок:
    • — инструкция с таблицами дозировок
    • — инструкция без таблиц дозировок
  • Кликните по названию препарата, чтобы открыть подробную информацию.
  • Для возврата к поиску используйте кнопку «Назад».
  • Если вы нашли ошибку, воспользуйтесь кнопкой ⚠️ для отправки сообщения.

Приложение находится в разработке. Если что-то работает не так — напишите нам!
